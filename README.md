# Smart Home Intrusion Detection System

## Project Overview
This project implements a smart home security system using multiple modules:
- Deep Learning (DL)
- Cloud Computing (CC)
- Software Engineering (SE)
- Network Security (NS)

## Network Security Module (NS)
- Device authentication
- Data encryption
- Anomaly detection
- Secure API communication

## Status
Phase 1 and Phase 2 of the Network Security module are completed.
